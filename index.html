O mapa não carregou

O Google recusou a chave da API configurada. É sempre um destes 4 motivos — confira na ordem:

  1. Chave errada ou com espaço — reabra Configurações e cole a chave de novo, sem espaços antes/depois.
  2. Referenciador HTTP não corresponde — no Google Cloud Console, a restrição da chave precisa ser exatamente a URL do seu site + /* no final.
  3. API não ativada — confira se Maps JavaScript API, Places API, Directions API e Geocoding API estão todas ativadas.
  4. Faturamento não vinculado — o projeto no Cloud Console precisa ter um cartão cadastrado, mesmo usando só a cota gratuita.
